. What is the "<!DOCTYPE html>"-declatation?
	-This declares and specifies this doc to be an html5 file.
	-The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages     correctly. This only needs to be written into the html doc, one time.

. ELEMENTS
	-
. What is the <html> element?
	-This is the "root" element of the html5 page.
. What is the <head> element?
	-This contains "MetaData" (meta information), about the document.
. What is the <title> element?
	-This specifies the/a title for the document.
. What is the <body> element?
	-This specifies the/a body, or visible content on the page.
. What is the <h1> element?
	-This is used to introduce your first/or multiple heading/s on the page, as well as, introducing the first heading on a page.
. What is the <p> element?
	-"<p>" is an element that defines a paragraph.


. HTML Tags
	
    -HTML tags normally come in pairs like <p> and </p>
    -The first tag in a pair is the start tag, the second tag is the end tag
    -The end tag is written like the start tag, but with a forward slash inserted before the tag name
    
    . Tip: The start tag is also called the opening tag, and the end tag the closing tag.

. HTML Documents

-All HTML documents must start with a document type declaration: <!DOCTYPE html>.

-The HTML document itself begins with <html> and ends with </html>.

-The visible part of the HTML document is between <body> and </body>.
